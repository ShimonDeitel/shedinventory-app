Privacy Policy

Shed Inventory is an offline-first app. All data you enter is stored locally on your device using Application Support storage and is never transmitted to any server. We do not collect, sell, or share any personal data. The app makes no network requests except standard Apple StoreKit calls required to process in-app purchases and restore purchases.

If you use Face ID / Touch ID to lock certain entries, authentication is handled entirely by iOS on-device; we never see or store biometric data.
